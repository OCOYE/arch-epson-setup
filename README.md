# Arch Epson Setup

A tiny script written in bash to install and configure all epson drivers to archlinux

# Installation
Before to install this script, you need the **YAY** installed in your system.
Link here to install **YAY**: https://github.com/Jguer/yay

Execute this command in your terminal to install:
<pre>
git clone https://github.com/OCOYE/arch-epson-setup.git
</pre>
After the installation, acess the folder and execute the ``start.sh`` script:
<pre>
  cd arch-epson-setup
  ./start.sh
</pre>
